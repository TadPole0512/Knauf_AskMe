🖥️ Memory Optimizer v2.0

Windows 메모리 자동 최적화 프로그램

메모리 사용률 90% → 70% 자동 관리

⚡ 빠른 설치 (3분 완료)

📥 Step 1: 압축 파일 다운로드 및 압축 해제

  1. MemoryOptimizer_v2.0.zip 파일 다운로드
  2. 원하는 위치에 압축 해제 (예: C:\Tools\)

⚙️ Step 2: 설치 스크립트 실행

⚠️ 중요: 관리자 권한으로 실행해야 합니다!
  1. install.bat 파일 우클릭
  2. "관리자 권한으로 실행" 선택
  3. 설치 완료 후 바탕화면에 바로가기 생성됨
✅ 설치 완료! 이제 메모리 최적화를 시작할 수 있습니다.

🚀 사용 방법

방법 1: 긴급 수동 정리 (즉시 실행) 🔥

권장 메모리가 부족할 때 바로 실행

실행 방법:

  • 바탕화면의 "메모리 긴급 정리" 아이콘 더블클릭

또는 PowerShell에서:

cd "C:\Program Files\MemoryOptimizer\scripts"
.\QuickMemoryClean.ps1
예상 효과: 2~4GB 메모리 확보 (약 20초 소요)

방법 2: 자동화 엔진 (백그라운드 실행) 🤖

고급 10초마다 자동 감시 및 최적화

⚠️ 사전 요구사항: Python 3.8+ 및 psutil 모듈 필요

Python 설치 여부 확인:

python --version
# Python 3.8 이상이면 OK

psutil 설치:

pip install psutil --break-system-packages

자동화 엔진 실행:

cd "C:\Program Files\MemoryOptimizer\scripts"
python memory_optimizer.py
동작 방식:
- 90% 이상: Level 1 자동 최적화
- 95% 이상: Level 2 중간 최적화
- 99% 이상: Level 3 긴급 모드

📊 최적화 레벨 설명

레벨 트리거 작업 내용 예상 효과
Level 1 메모리 90% 이상 • 임시 파일 삭제
• DNS 캐시 플러시
• 휴지통 비우기 		300~800MB
Level 2 메모리 95% 이상 • Level 1 전체 실행
• 프로세스 메모리 압축
• 백그라운드 프로세스 우선순위 하향 		500MB~1.5GB
Level 3 메모리 99% 이상 • Level 2 전체 실행
• 고메모리 프로세스 종료 (화이트리스트 제외)
• 긴급 알림 표시 		1~3GB

🛡️ 보호 프로세스 (화이트리스트)

절대 종료되지 않는 프로세스:
Visual Studio Code
IntelliJ IDEA
DBeaver
Chrome
Edge
Teams
Remote Desktop
Explorer

프로세스 추가 방법

  1. C:\Program Files\MemoryOptimizer\config\config.json 파일 열기
  2. "whitelist" → "processes" 배열에 프로세스 이름 추가
  3. 예시: "your_app.exe"

🛠️ 문제 해결

❌ 문제: "관리자 권한이 필요합니다"

해결:

  1. 스크립트 파일 우클릭
  2. "관리자 권한으로 실행" 선택

❌ 문제: "psutil 모듈을 찾을 수 없음"

해결:

pip install psutil --break-system-packages

❌ 문제: 최적화 후에도 메모리 90% 유지

해결 방법:

  1. 작업 관리자 열기 (Ctrl + Shift + Esc)
  2. 메모리 탭에서 사용량 많은 프로세스 확인
  3. 불필요한 프로그램 수동 종료
  4. Chrome 탭 많은 경우: 탭 정리
  5. 최후 수단: 컴퓨터 재시작

🚨 긴급 복구 필요 시

복구 스크립트 실행:

  1. 시작 메뉴 → Memory Optimizer → 긴급 복구

또는 수동 실행:

"C:\Program Files\MemoryOptimizer\scripts\restore.bat"
복구 스크립트는 다음 작업을 수행합니다:
• Windows Update 서비스 재시작
• SuperFetch/SysMain 복원
• Windows Search 복원
• 시스템 파일 무결성 검사

🔄 Windows 부팅 시 자동 실행 설정

작업 스케줄러 사용

  1. 작업 스케줄러 열기 (Win + R → taskschd.msc)
  2. "기본 작업 만들기" 클릭
  3. 트리거: Windows 시작 시
  4. 동작: 프로그램 시작
    • 프로그램: python.exe
    • 인수: C:\Program Files\MemoryOptimizer\scripts\memory_optimizer.py
  5. "가장 높은 권한으로 실행" 체크
  6. 완료!
